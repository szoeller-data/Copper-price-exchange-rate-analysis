# Copper-price-exchange-rate-analysis
Time series analysis of copper prices and exchange rate dynamics in Peru using a VARX model in R

## Project Overview
This project analyzes how international copper price shocks affect the real effective exchange rate (REER) and international reserves accumulation in Peru.

The analysis is based on quarterly data from 2002 to 2024 and explores whether commodity price increases lead to currency appreciation.

## Methodology
- Time series analysis
- VARX model (Vector Autoregression with exogenous variables)
- Stationarity testing (ADF test)
- Dynamic multipliers
- Cumulative Effects
- Bootstrap confidence intervals

Tools:
- R (RStudio)

## Key Results
- Copper price shocks create moderate appreciation pressure on exchange rates  
- Effects are relatively small and not statistically significant  
- International reserves respond more strongly than exchange rates  
- Suggests central bank intervention absorbs part of external shocks
